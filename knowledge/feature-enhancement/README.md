# 系统功能增强 Knowledge

本目录记录“系统原本能工作，但需要新增能力、改变默认行为或本地客制化”的知识。目录按场景分类，与 `references/feature-enhancement/` 一一对应。

## 场景

- `system/`：维护流程和持久化策略增强。
- `applications/`：应用体验和桌面入口增强。
- `ukui/`：UKUI 全局搜索、搜索引擎、自定义命令、源码级桌面增强。
- `network/`：网络策略和代理集成增强。
- `hardware/`：硬件能力启用后的增强记录。
- `storage/`：DATA 工作区和本地开发空间策略。
- `agent-tools/`：AI 工具全局提示词、权限和自动加载规则。
- `source-rebuild/`：源码级功能增强、本地 patch、构建和清理策略。
